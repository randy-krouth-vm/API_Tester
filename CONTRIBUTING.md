# Contributing

Thanks for contributing.

## Rules
- Keep changes scoped and testable.
- Do not add offensive/persistent attack behavior.
- Keep all testing features explicitly defensive and authorized-use focused.

## Build
```powershell
dotnet build API_Tester.slnx -nologo
```

## Pull Requests
- Describe what changed and why.
- Include screenshots for UI changes.
- Include any behavior/risk tradeoffs.
